## 快速背景

PCVR 任务的数据集为 Parquet 格式，120 列：

| 列类型           | 数量 | 说明                                                       |
| ---------------- | ---- | ---------------------------------------------------------- |
| ID/标签          | 5    | user_id, item_id, label_type, label_action_type, timestamp |
| user_int_feats   | 46   | 用户整数特征                                               |
| user_dense_feats | 10   | 用户稠密特征                                               |
| item_int_feats   | 14   | 物品整数特征                                               |
| 序列特征         | 45   | 4 个域（a, b, c, d）的序列数据                             |
